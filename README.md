# 🏥 AI Healthcare Prediction System

## 📌 Overview

The AI Healthcare Prediction System is a full-stack web application that predicts potential health conditions based on user-provided medical information. The system uses Artificial Intelligence concepts to assist users in obtaining an early health assessment. It provides an easy-to-use interface for entering patient details and displaying prediction results.

This project was developed as an academic project to demonstrate full-stack web development, database integration, and healthcare applications.

---

## 🚀 Features

- User-friendly healthcare prediction interface
- Patient information management
- AI-based health prediction
- Secure data storage
- Responsive user interface
- REST API integration
- Database connectivity
- Real-time prediction results

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/kalaiselvi58642-ship-it/AI-Healthcare-Prediction.git
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

### Backend

1. Open the Spring Boot project in Eclipse or IntelliJ IDEA.
2. Configure the MySQL database.
3. Update the database credentials in `application.properties`.
4. Run the Spring Boot application.
